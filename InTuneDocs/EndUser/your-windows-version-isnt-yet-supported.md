# <a name="your-windows-devices-operating-system-version-isnt-yet-supported"></a>Windows cihazınızın işletim sistemi sürümü henüz desteklenmiyor

Mobil teknolojilerin hızlı gelişimi nedeniyle, bazı durumlarda cihazınız, BT uzmanınızın sınadığı cihazdan daha gelişmiş olabilir. En yeni Windows sürümü, şirketinizde kullanılan bazı diğer araçlar ile çalışmayabilir.

Bu sorunu çözmek için BT yöneticinize başvurmanız gerekecektir. Kişi bilgileri için [Şirket Portalı Web sitesine](http://portal.manage.microsoft.com) bakın.
