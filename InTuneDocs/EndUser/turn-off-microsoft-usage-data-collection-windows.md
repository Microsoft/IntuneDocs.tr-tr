# Microsoft kullanım verilerini toplamayı devre dışı bırakma

Microsoft, ürün ve hizmetlerini geliştirmek için Şirket Portalı uygulamasının güvenilirliği, performansı ve nasıl kullanıldığı hakkında anonim verileri otomatik olarak toplar.

Şirket Portalı uygulamasındaki **Kullanım Verileri** ayarını kullanarak Windows cihazınızda söz konusu verilerin toplanmasını kapatabilirsiniz. BT yöneticileri bu verilerin toplanması üzerinde denetim sahibi değildir ve bu ayar için yaptığınız seçimi değiştiremez.

Bu bilgiler yardımcı olmadı mı? BT yöneticinize başvurun. Kişi bilgileri için [Şirket Portalı Web sitesine](http://portal.manage.microsoft.com) bakın.
