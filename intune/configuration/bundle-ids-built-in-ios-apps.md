# <a name="bundle-ids-for-built-in-ios-apps-you-can-use-in-intune"></a>Bundle IDs for built-in iOS apps you can use in Intune

When you configure features on iOS devices, you can also add the built-in apps on iOS devices. This article lists the bundle IDs of some common built-in iOS apps. Diğer uygulamaların paket kimliğini bulmak için yazılım satıcınıza başvurun. See Apple's list of [iOS bundle IDs](https://support.apple.com/guide/mdm/ios-bundle-ids-mdm90f60c1ce/web) (opens Apple's web site).
